# Pilot Demo Script

## Goal
Show that the system increases qualified lead capture and creates advisor-ready handoffs.

## Setup
- Use a pilot tenant with real agency knowledge documents ingested.
- Ensure widget domain and token flow are active.
- Open CRM test workspace for handoff verification.

## Scripted journeys
1. Luxury honeymoon
- User asks for Tahiti/Bora Bora style trip.
- Bot discovers intent and qualification fields.
- Bot returns grounded options and citation-backed rationale.
- Bot escalates with concierge-style assumptive close.

2. Family trip
- User has mixed priorities (budget + kid-friendly + date constraints).
- Bot collects missing slots, refines options, and proposes next-best CTA.

3. Flexible destination traveler
- User asks for ideas rather than exact destination.
- Bot narrows via preferences and seasonality.
- Bot produces shortlist and advisor handoff trigger.

## What to measure live
- Time to qualification completeness.
- Recommendation relevance perception by advisor.
- Handoff payload quality (usable without manual rewrite).
- Drop-off points in the funnel.

## Honest framing for demos
- Clearly separate what is live today versus roadmap.
- Prioritize trust and transparency over over-promising.
- End each demo with one pilot KPI target and implementation timeline.
