# Travel AI Pilot Foundation

Last updated: 2026-05-17

This folder contains a complete docs package for a services-led, B2B luxury travel AI concierge platform.

## Files
- `01-product-overview.md`: Product definition, ICP, value proposition.
- `02-system-architecture.md`: Technical architecture and API contracts.
- `03-go-to-market-and-viability.md`: Business viability and VC-style analysis.
- `04-competitive-landscape.md`: Competitor map and strategic implications.
- `05-go-live-checklist.md`: Human + engineering launch checklist.
- `06-demo-script.md`: Guided pilot demo script.

## Positioning
This product is a conversion engine for luxury travel agencies:
- Improve lead intake and qualification quality from website traffic.
- Route high-intent leads into advisor workflows with contextual handoff.
- Use agency-specific RAG to ground recommendations in real offers and policies.

## Current state summary
- Strong MVP foundation exists.
- Codebase is demo-ready for guided pilots.
- Main gaps before paid production pilots are real CRM integration depth, production semantic embeddings enforcement, and onboarding execution discipline.
